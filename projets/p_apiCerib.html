API NodeJS - Cerib

API NodeJS Restful

Cette API qui fut créée en 1 mois et demi a pour but de recevoir les requêtes des clients qui recherchent des contacts ou des tiers dans l'entreprise. L'API renvoie les données recherchées via des requêtes SQL.

Ce projet a été initié pour remplacer un existant qui n'etait pas assez intuitif et avec des problèmes d'envoie à la base de données.
